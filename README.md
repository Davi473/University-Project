# University-Project

Projeto Java minimal usando Maven.

Como usar:

- Compilar e rodar testes:

```bash
mvn test
```

- Empacotar:

```bash
mvn package
```

- Executar (após `mvn package`):

```bash
java -cp target/university-project-0.1.0.jar com.example.App
```

Notas:
- O projeto usa Java 17 e Maven por padrão. Se preferir Gradle ou outra versão do Java, me diga que eu ajusto.
